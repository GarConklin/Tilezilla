Menu hit-area tuner

Colored boxes show each menu link. Green = close (X) — returns to the game. Screenshot this page or use Copy report for chat and paste into Cursor so link positions can be aligned. Saves to data/menu_layout.json. Back to player select

Controls

—

  • Link selected: Scroll — Y% · Shift+scroll — X% · Ctrl+scroll — width% · Alt+scroll — height%
  • Close (green X): Scroll — Y · Shift — X · Ctrl — W · Alt — H
  • Menu width selected: Alt+scroll — stretch wider/narrower · Alt+Shift — move up/down
  • Ctrl+Shift / Alt+Ctrl+scroll — default row size (new links only)
