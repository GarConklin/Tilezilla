Tilezilla -  Puzzle Game – Progression Strategy Document
Project Overview

This is a mobile-friendly browser-based puzzle game built around rotating connected two-square tiles to create valid paths between a Start and End tile.

The game is designed to:

be easy to learn
gradually increase in complexity
support endless puzzle generation
encourage replayability
work well on phones and desktop browsers

The project is intended to become:

a Progressive Web App (PWA)
playable directly in browser
hosted online
account/database driven later for progression tracking
Core Gameplay
Tile Structure

Each tile:

consists of two connected squares:
A
B
contains path/hotspot edges
can rotate clockwise
hotspots rotate with the tile

Rotation logic:

N → E → S → W → N

Tiles must:

connect valid live edges
avoid invalid edge collisions
avoid live edges pointing off-board unless allowed
