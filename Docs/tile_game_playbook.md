# Tile Puzzle Game Playbook

## Core Game

Place AB tiles to complete a valid board.

## Board System

Sizes: 2x4 to 6x6\
Odd grids → blocker → later replaced with E1 tile (player places gap)

## Difficulty

Based on: - size - solve_count

Buckets: - Easy: many solutions - Medium: moderate - Hard: few - Expert:
1 solution

## Scoring

Moves = number of placements

## Level Goals

-   Complete
-   Efficient (target moves)
-   Perfect (best)

## Multi-Solution System

-   Find 2nd solution
-   Find multiple solutions
-   Must differ by 4--5 tiles

## Super Challenge

Find all solutions (tiered goals)

## E1 Tile System

Replace blocker → player decides gap position

## Rotation Variants

Use 0°, 90°, (maybe 270°)

## Progression

Bronze → Silver → Gold → Master

## Time

Par time per level

## Hints

Earn via play, spend for help\
Does not affect XP

## XP

From performance only

## Daily Challenge

1 puzzle per day

## Modes

Campaign, Daily, Endless, Challenges

## Monetization

Buy hint tokens (no pay-to-win)

## Key Principle

Build a level selection system, not a level list

## Build Order

1.  Core play
2.  Moves + completion
3.  Small progression
4.  Multi-solution
5.  Daily
6.  E1
7.  Super challenges
