How to run (Windows 11)
Start Docker Desktop.
In PowerShell from the repo root:
.\scripts\start-remote-test.ps1 -OpenFirewall -SeedUsers
On your phone/tablet (same Wi‑Fi), open the LAN URL the script prints, e.g. http://192.168.1.42:3000.
Stop:

.\scripts\stop-remote-test.ps1
What you get
One URL for everything (game + auth API)
Guest daily challenge works without login (best for remote UI testing)
-OpenFirewall adds a Windows inbound rule for the port
-SeedUsers loads gar/gar and Arn/arn into the local auth DB
README.md has a short remote-testing section as well.

Note: If other devices can’t connect, run again with -OpenFirewall, or allow the port manually. Guest play is the main remote-test path; full registered login still depends on your auth API wiring. Say if you want this committed to git.`
